1) Types of Neural Networks
	- Artificial Neural Network ANN - Also called MultiLayer Perceptron MLP
	- Convolution Neural Network CNN - For Images
	- Recurring Neural Network RNN - For Time Series Like Stock Prices
	- Recursive Neural Tensor Network - To analyze hierarchical structured data

2) Libraries
	Research Oriented
	- Tensorflow
	- Theano
	
	Commercial Grade
	- CNTK
	- Torch - C/Lua
	- Caffe - C++ Library
	- DeepLearning4j - Java Library
	
3) Platforms for DeepLearning
	- H20
	- Dato GraphLab	

4) Initializers - Used to initialize the weights in some manner
	Well Known
		- Zeros
		- Ones
		- Constant
		- RandomNormal
		- RandomUniform
	
5) Activation Functions
	Well Known
		- Linear
		- Sigmoid
		- Hyperbolic Tangent (tanh)
		- Rectified Linear Unit (ReLU)
			
6) Losses
	Basic: 
		- mean_squared_error
		- mean_absolute_error
		- mean_absolute_percentage_error
		- mean_absolute_logarithmic_error
		- binary_crossentropy
		- categorical_crossentropy

7) Optimizer 
	Basic:
		- SGD (Stochastic Gradient Descent)
		
8) Metrics
	Basic:
		- accuracy -- binary_accuracy
				   -- categorical_accuracy
				   -- sparse_categorical_accuracy
				   -- top_k_categorical_accuracy
				   -- sparse_top_k_categorical_accuracy
